**Это мини-оболочка командной строки, реализующая основные shell-команды для работы с файловой системой: навигация, просмотр, копирование, перемещение, удаление и чтение файлов.**
**В проекте использованы стандартные встроенные библиотеки (os, pathlib, shlex, shutil), а так же модуль pytest для тестирования**

### Поддерживаемые команды:

- `ls` `[-l]`: Показывает список файлов и папок в текущей директории. Флаг -l выводит подробную информацию (права, размер, дата, название).
- `cd` `<путь>`: Меняет текущую рабочую директорию.
- `cp` `[-r]` `<источник> <назначение>`: Копирует файл или каталог. С флагом -r рекурсивно копирует папки.
- `mv` `<источник> <назначение>`: Перемещает или переименовывает файл/папку.
- `rm` `[-r]` `<путь>`: Удаляет файл или каталог. Флаг -r позволяет рекурсивно удалять каталоги.
- `cat` `<файл>`: Показывает содержимое файла.
- `--help` `<команда>`: Краткая справка по командам.
- `exit`: Завершает работу.

### Особенности

- Команды `ls`, `cat` принимают на вход несколько аргументов и делают поочередный вывод с названием каждого файла.
- Команды `cp`, `mv`, `rm` принимают на вход несколько аргументов, но последний обязательно должен быть директорией.
- Команда `cd` поддерживает аргумент `..` для перехода на уровень выше, и `~` либо пустой символ (`cd `) для перехода в домашнюю директорию.
- Весь журнал операций сохраняется в файл shell.log с помощью модуля logging.

### Требования
- Python 3.10 или выше
- Установленные пакеты: pytest

### Запуск программы

```bash
python main.py
```

### Ограничения:
- если для `cp` (или `mv`) передается несколько аргументов, то последний обязательно должен быть директорией, в которую будут копироваться (переноситься) предыдущие файлы и каталоги.
- запрещено удалять корневую или домашнюю директорию командой `rm [-r]`.
